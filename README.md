Product Manager's Questions

## Contributing

Contributions are welcome. This repository is issue-driven: open a GitHub issue
describing the change you'd like to make, then submit a pull request that
references it. Keep pull requests focused on a single issue and describe what
you verified. See [AGENTS.md](AGENTS.md) for the automated workflow that drives
issues and pull requests through review.
